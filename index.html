03. Skills & Expertise

Technical Skills

My development journey is built on a strong foundation of HTML5 and CSS3, crafting semantic, accessible markup with modern layouts using Flexbox and Grid. I bring interfaces to life with JavaScript (ES6+), building dynamic, interactive experiences. On the frontend, I work extensively with React.js, leveraging hooks, context API, and component-based architecture to create scalable applications.

On the backend, I build robust APIs and server-side applications using Node.js and Express.js, with data persistence handled through MongoDB and PostgreSQL. I'm experienced with RESTful API design, authentication systems, and real-time communication using WebSockets.

My toolkit includes Git for version control, VS Code as my primary editor, and Figma for design collaboration. I work comfortably with Docker for containerization, Postman for API testing, and Firebase for rapid prototyping and deployment.
